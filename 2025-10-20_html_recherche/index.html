1. Grundstruktur

<!DOCTYPE html>
<html lang="de">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Seite</title>
    </head>
    <body>
        ...
    </body>
</html>

2. Textstrukturierung

  • <h1> bis <h6> - Überschriften
  • <p> - Absatz
  • <br> - Zeilenumbruch
  • <hr> - Horizontale Linie
  • <strong> - Wichtiger Text
  • <em> - Betonter Text (kursiv)

3. Links und Navigation

<a href="https://beispiel.at" target="_blank">Link</a>
<nav>
    <a href="/seite1">Seite 1</a>
    <a href="/seite2">Seite 2</a>
</nav>

4. Listen

<ul>
    <li>Unordered Item</li>
</ul>

<ol>
    <li>Ordered Item</li>
</ol>

5. Bilder und Medien

<img src="bild.jpg" alt="Beschreibung" width="300">

<figure>
    <img src="bild.jpg">
    <figcaption>Bildunterschrift</figcaption>
</figure>

6. Tabellen

<table>
    <thead>
        <tr><th>Spalte 1</th></tr>
    </thead>
    <tbody>
        <tr><td>Daten</td></tr>
    </tbody>
</table>

7. Formulare

<form action="/submit" method="POST">
    <label for="name">Name:</label>
    <input type="text" id="name" name="name" required>
    
    <label for="email">E-Mail:</label>
    <input type="email" id="email" name="email">
    
    <button type="submit">Absenden</button>
</form>

8. Wichtige Attribute

  • class - CSS-Klassen zuweisen
  • id - Eindeutige ID
  • style - Inline-Styles
  • title - Tooltip beim Hover
  • data-* - Benutzerdefinierte Daten

Fazit

HTML5 bietet viele semantische Tags die wichtig sind für:

  • Barrierefreiheit
  • SEO (Suchmaschinen)
  • Code-Wartbarkeit
